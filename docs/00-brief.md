# 00 — MVP Brief

## MVP 한 줄 요약

> 패스트캠퍼스 B2B 마케팅팀이 GEO(Generative Engine Optimization)를 **선택지 기반 가이드 멘토링**으로 학습하는 웹 시스템

## 제약조건

- 스택: HTML/CSS/JS (바닐라) + FastAPI + SQLite
- LLM 사용 없음 (영구) — 모든 콘텐츠와 선택지를 사전 작성하여 seed 데이터로 관리
- 인터랙션: 자유 텍스트 입력 없음, 미리 준비한 옵션을 선택하며 진행
- MVP 범위: Stage 1(GEO 세계관 이해) + Stage 2(Technical GEO 기초 실습), 총 16개 모듈
- 콘텐츠 소스: materials/ 폴더 내 3개 문서에서 직접 큐레이션

## 가정

- [x] 사용자는 패스트캠퍼스 B2B 마케팅팀 (GEO 초보자)
- [x] 선택지 기반만으로도 인지적 도제 원리(모델링/코칭/스캐폴딩/반성)를 구현 가능
- [x] 단일 사용자(default) 진도 관리로 충분 (인증 불필요)
- [ ] 16개 모듈 × 평균 5 steps의 seed 데이터가 충분한 학습량을 제공하는가
- [ ] Stage 1 종합평가 70% 기준이 적절한 난이도인가
